Collection of free software for Windows

Posted on Mon 15 January 2018 in Windows • 2 min read

  • Yawcam - is a webcam software for Windows written in Java.

  • iSpy - is our free, open source video surveillance platform.

  • WinSCP - WinSCP is an open source free SFTP client, FTP client, WebDAV client and SCP client for Windows.

  • KiTTY - KiTTY is a fork from version 0.67 of PuTTY, the best telnet / SSH client in the world. KiTTY is only designed for the Microsoft Windows platform.

  • FileZilla - the free FTP solution. Both a client and a server are available.

  • Everything - Locate files and folders by name instantly. You can use everything to run a readonly ftp or http server that will share your data.

  • Inkscape - is a professional vector graphics editor for Windows, Mac OS X and Linux. It's free and open source.

  • VirtualDub - is a video capture/processing utility for 32-bit and 64-bit Windows platforms (98/ME/NT4/2000/XP/Vista/7)

  • Open Hardware Monitor - is a free open source software that monitors temperature sensors, fan speeds, voltages, load and clock speeds of a computer.

  • Notepad++ - is a free (as in "free speech" and also as in "free beer") source code editor and Notepad replacement that supports several languages.

  • Calibre - is a free and open source e-book library management application developed by users of e-books for users of e-books. It has a cornucopia of features divided into the following main categories:

  • 7-Zip - is a file archiver with a high compression ratio.

  • Squeezelite - lightweight headless Squeezebox emulator

  • Rufus - Create bootable USB drives the easy way

  • Win32 Disk Imager - This program is designed to write a raw disk image to a removable device or backup a removable device to a raw image file.

  • FAT32 Format - Format parttions bigger than 32 GB with FAT32

  • SD Card Formatter - The SD Card Formatter was created specifically for memory cards using the SD/SDHC/SDXC standards.

  • MiniTool Partition Wizard Free Edition - a free partition manager software designed by MiniTool Solution Ltd

  • Cmdow - Win32 commandline utility for NT4/2000/XP/2003/2008/7 that allows windows to be listed, moved, resized, renamed, hidden/unhidden, disabled/enabled, minimized, maximized, restored, activated/inactivated, closed, killed and more.

  • NSSM - NSSM is a service helper program similar to srvany and cygrunsrv. It can start any application as an NT service and will restart the service if it fails for any reason.

  • Minimal ADB and Fastboot Tool - The total installation of Minimal ADB and Fastboot tool is around 2 mb only (where as android sdk package takes more than 500 mb).
